3. Release¶

Para gerar uma nova versão, alguns passos são necessários:

  • Adicione uma entrada ao CHANGES.rst seguindo o mesmo padrão dos anteriores

Exemplo:

0.1.0
-----

- What was changed

  • Reveja as alterações feita nas dependências requirements.txt

  • Commit e faça o push das alterações

  • Rode o comando para gerar a release que irá rodar os tests, compilar e fazer o upload para o Pypi

$ make release

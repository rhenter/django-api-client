Dependências¶

  • Python 3.x

  • Django 2.0 ou posterior

Como instalar¶

Você pode instalar o Django API Client usando pip:

$ pip install django-api-client

Se você quer instalar a partir do código fonte, pegue o repositório do GitHub em rode o setup.py

$ git clone git@github.com:rhenter/django_api_client.git
$ cd django_api_client
$ python setup.py install

Como configurar¶

Para habilitar o django_api_client você precisa adiciona-lo ao INSTALLED_APPS no seu arquivo settings.py do seu projeto:

INSTALLED_APPS = (
    ...
    'django_api_client',
    ...
)
